
#  100 Days of Code Challenge

Welcome to my **#100DaysOfCode** journey!
This repository documents my commitment to coding every day, improving my skills, and building consistency as a software engineer.

## Challenge Goal

The goal of this challenge is simple:

* Code **at least every day for 100 days**
* Stay consistent and accountable
* Improve problem-solving and development skills
* Build real-world projects and deepen understanding

##  What I’m Learning

Throughout this journey, I’ll be focusing on:

* Core programming fundamentals
* Data structures & algorithms
* Frontend and/or backend development
* Building real-world projects
* Writing clean, maintainable code
