# 100 Days of Code — Phase 1 (Days 1–30)

## Project-Based Learning Approach

This phase is structured around building **3 real-world projects**, each completed in **10 days**.

Instead of random practice, each project is built **section by section**, just like real-world development.

---

# Tech Stack

- HTML5
- CSS3
- SCSS
- TailwindCSS
- JavaScript

---

# PHASE 1 GOAL

By the end of 30 days, I will:

- Build responsive websites from scratch
- Understand layout systems (Flexbox & Grid)
- Write scalable styles using SCSS
- Use TailwindCSS efficiently
- Add interactivity using JavaScript
- Have **3 complete portfolio-ready projects**

---

* Build everything from memory first
* Debug your own errors before seeking help
* Focus on understanding, not speed
